First you need to install the keycloak-angular package, which is a wrapper around the keycloak-js library, which is what you use to access the Keycloak APIs.

npm install @keycloak/keycloak-angular

Once you have the keycloak-angular package installed, you can use it in your Angular application to access the Keycloak APIs.
